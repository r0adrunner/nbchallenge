# NBGraph

NB Challenge

## Requisites

* Leiningen 2

* Java 1.7+

* MongoDB

MongoDB Should be up on localhost on default port (27017)

## Usage

First, you should download any dependencies

    $ lein deps

To run tests

    $ lein test

Run with

    $ lein ring server

The app should be accessible on http://localhost:3000/
